# SYKRON — Portfolio

Personal portfolio site for **Sykron** — dev / programmer.

Y2K Japanese magazine + Jet Set Radio street aesthetic. Chrome 3D orbs, PSX vertex jitter, halftone screentone, spray-paint splats, heavy outlined block type.

Showcases:
- **RAMMY** — Windows RAM optimizer with live EKG visualizer
- **TinyPic** — instant screenshot + clipboard utility

## Stack

React 19 · Vite · CSS

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```
